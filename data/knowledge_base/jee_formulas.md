# JEE Algebra Identities

- **Quadratic Formula**: The roots of $ax^2 + bx + c = 0$ are $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$.
- **Binomial Theorem**: $(a+b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k}b^k$.
- **Arithmetic Progression (AP)**: nth term $a_n = a + (n-1)d$, Sum $S_n = \frac{n}{2}[2a + (n-1)d]$.
- **Logarithms**: $\log_a(mn) = \log_a(m) + \log_a(n)$, $\log_a(m^n) = n\log_a(m)$.

# JEE Calculus Rules

- **Power Rule**: $\frac{d}{dx}(x^n) = nx^{n-1}$.
- **Product Rule**: $\frac{d}{dx}(uv) = u\frac{dv}{dx} + v\frac{du}{dx}$.
- **Chain Rule**: $\frac{d}{dx}(f(g(x))) = f'(g(x))g'(x)$.
- **Integration by Parts**: $\int u \, dv = uv - \int v \, du$.

# JEE Probability Formulas

- **Bayes' Theorem**: $P(A|B) = \frac{P(B|A)P(A)}{P(B)}$.
- **Independent Events**: $P(A \cap B) = P(A)P(B)$.
- **Binomial Distribution**: $P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$.
